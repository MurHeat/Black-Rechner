MurHeat BLACK – Heizflächen‑Rechner (A3 Carbon Blue, animiert, i18n)

Schnell‑Hosting:
- **Netlify Drop**: https://app.netlify.com/drop → Ordnerinhalt ziehen → sofortige HTTPS‑URL.
- **Vercel**: Neues Projekt → Import als „Other“ → Ordner hochladen → URL.
- **GitHub Pages**: Repo erstellen → Dateien committen → Settings > Pages > Deploy from Branch.

Installation (iPhone):
- Safari öffnen → URL aufrufen → „Teilen“ → „Zum Home‑Bildschirm“. Offline dank Service Worker.

Hinweis: Sprache oben rechts umschalten, Auswahl wird lokal gespeichert.
